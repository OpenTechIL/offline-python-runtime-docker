Wheelhouse is concept of folder that has whl files for use in offline airgapped env

the wheelhouse located at /lib/python_wheelhouse

Structure:
 `requirements-global.txt` - installs to default python
 `requirements-wheelhouse.txt` - download whl files to folder to use
 `requirements.txt` - what sould installed in the app from the wheelhouse, 
    Using like `pip install --no-index --find-links=/lib/python_wheelhouse -r requirements.txt`


for development:
`podman build . -t offline-python-runtime:dev`

new features:
* Added levels of installation:
    - Global scope- install globaly
    - wheelhouse socpe - download only for pip install 
    - local - will install from wheelhouse to user appuser
* Improved size
* use global pip config to use wheelhouse
